Task 2
Part 1 – Very Easy (Foundations)
Create a basic HTML document with the correct structure:

<!DOCTYPE html>

<html>, <head>, <title>, <body>

Inside the <body>, add:

One <h1> heading with the text: "My First HTML Page"

One paragraph <p> describing yourself in 2–3 sentences.

Add the following headings in order:

<h2> – "About Me"

<h3> – "My Hobbies"

<h4> – "My Favorite Books"

Part 2 – Easy (Text Formatting and Structure)
Create a paragraph that contains:

One bold word

One italic word

One <u>underlined</u> word

Add a horizontal line <hr> between two sections of your page.

Create two paragraphs:

The first paragraph should contain a line break using <br>

The second paragraph should not use <br>

Write a comment in your HTML code explaining what the page is about.

Part 3 – Medium (Lists and Links)
Create an unordered list of at least 5 hobbies.

Create an ordered list of at least 5 steps to learn programming.

Create a nested list:

A main list of 3 subjects (e.g., Programming, Mathematics, Physics)

Each subject should have a sublist with at least 2 topics.

Add a link (<a>) to the W3Schools HTML tutorial:

The link text should be: "Learn HTML on W3Schools"

The link should open in a new tab.

Add an email link using mailto: with your own email address.

Part 4 – Medium+ (Images and Basic Attributes)
Add an image using <img>:

Set src, alt, width, and height attributes.

Place the image:

Between two paragraphs.

Add a caption using a paragraph below the image.

Create a link that wraps around an image (clicking the image opens a website).

Part 5 – Hard (Tables – Basic)
Create a table with:

3 columns: Name, Age, City

4 rows of data.

Add a header row using <th>.

Add a table caption using <caption>.

Set:

Table border

Cell padding

Cell spacing

Part 6 – Harder (Tables – Advanced Structure)
Create a table with:

<thead>, <tbody>, and <tfoot>

The table should represent Student Grades with columns:

Student Name

Subject

Midterm

Final

Total

Use colspan to merge two header cells into one.

Use rowspan to merge two cells in the first column.

Align:

Header text to the center

Numeric values to the right

Part 7 – Very Hard (Mini Project)
Create a complete HTML page called student_profile.html that contains:

Page title: Student Profile

Main heading: Student Profile Page

Section 1: Personal Information

Name, age, city in a small table.

Section 2: Education

An ordered list of schools/universities.

Section 3: Skills

An unordered list with at least 6 skills.

Section 4: Weekly Schedule

A table with:

Days of the week as columns

Time slots as rows

At least one merged cell using colspan or rowspan.

Add comments before each section explaining its purpose.
